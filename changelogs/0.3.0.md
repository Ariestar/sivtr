# 0.3.0 - 2026-07-21

## English

### Added

- Added Grok as a first-class agent provider (`~/.grok` session parse) and moved MCP install/uninstall onto a shared host registry so every provider shares one config path. ([f88364e](https://github.com/Ariestar/sivtr/commit/f88364e))
- OpenClaw sessions now read from per-agent SQLite (`session_entries` + `transcript_events`) with legacy JSONL fallback; Cursor reads real agent-transcript JSONL instead of a stub. ([88a7e64](https://github.com/Ariestar/sivtr/commit/88a7e64), [c43306c](https://github.com/Ariestar/sivtr/commit/c43306c))
- Hermes primary store is now `state.db` (JSONL residual); workspace filters keep unbound sessions that lack a cwd. ([e1024d4](https://github.com/Ariestar/sivtr/commit/e1024d4))
- Typed structure parts: skill / thinking / MCP channels as `<:channel:…:>` markers; reading mode folds structure to marker-only lines, raw expands full payloads. ([84af70d](https://github.com/Ariestar/sivtr/commit/84af70d), [6f5d34c](https://github.com/Ariestar/sivtr/commit/6f5d34c), [1c39dba](https://github.com/Ariestar/sivtr/commit/1c39dba))
- Browse TUI is a product entry (`sivtr` / hotkey / pick): multi-source `query_many`, non-blocking source loads, registry-driven help/footer keys. ([25641d0](https://github.com/Ariestar/sivtr/commit/25641d0), [1a2cfe5](https://github.com/Ariestar/sivtr/commit/1a2cfe5), [48b20d3](https://github.com/Ariestar/sivtr/commit/48b20d3))
- Content pane splits into independent Input / Output halves (Tab switch, mouse hit-test, free drag select / Ctrl-drag block select). Focus-biased heights and expandable source list when focused. ([d9e1e16](https://github.com/Ariestar/sivtr/commit/d9e1e16), [2ead582](https://github.com/Ariestar/sivtr/commit/2ead582), [b0cd671](https://github.com/Ariestar/sivtr/commit/b0cd671))
- Viewport sliding-window session load: meta grows from visible rows + prefetch; bodies stay only for focus / multi-select / neighbors and evict on scroll-away. ([3c7d118](https://github.com/Ariestar/sivtr/commit/3c7d118))
- Config-scoped `&alias` source expansion (`alias set|list|remove`); short aliases expand before load. ([0b8035b](https://github.com/Ariestar/sivtr/commit/0b8035b), [94087f1](https://github.com/Ariestar/sivtr/commit/94087f1))
- MCP install selects hosts with `-p/--provider` (detect installed hosts by default); OpenClaw host install writes into `~/.openclaw/openclaw.json`. ([45a5635](https://github.com/Ariestar/sivtr/commit/45a5635), [c43306c](https://github.com/Ariestar/sivtr/commit/c43306c))
- Doctor checks session storage under `data_dir` workspaces, can install the `sivtr-memory` skill, and `--fix` merges unique logs then drops legacy workspace-key duplicates. Setup also installs the skill when missing. ([4ac7e0d](https://github.com/Ariestar/sivtr/commit/4ac7e0d), [919bbaa](https://github.com/Ariestar/sivtr/commit/919bbaa))
- Copy agent entry is registry-driven (no per-provider clap variants); doctor splits agent-host detection from real sivtr MCP registration. ([1789aed](https://github.com/Ariestar/sivtr/commit/1789aed))

### Changed

- `WorkRef` is now `scope + path + at` (e.g. `desk:codex/abc/5/i/2`); legacy `Local`/`Remote` + `origin:body` helpers are gone. Mount aliases remain named scopes. ([a8f2eff](https://github.com/Ariestar/sivtr/commit/a8f2eff))
- Share invite is a separate step: bare `sivtr share` only creates the share; issue keys with `sivtr share invite <name>`. ([6cd1569](https://github.com/Ariestar/sivtr/commit/6cd1569), [01273b6](https://github.com/Ariestar/sivtr/commit/01273b6))
- Dropped `sivtr use` / workspace `context.toml`; keep explicit `desk:` / `&alias:` forms only. ([94087f1](https://github.com/Ariestar/sivtr/commit/94087f1))
- MCP install no longer accepts `auto` or `-t/--target`; use `-p/--provider` or default detect. ([45a5635](https://github.com/Ariestar/sivtr/commit/45a5635))
- Workspace origin CLI renamed to `sivtr ws` (was `wb`). ([a309b1c](https://github.com/Ariestar/sivtr/commit/a309b1c))
- Local and remote shares share one `query(source, Filter)` path; remote only forwards Query and applies the same filter on the share root. ([68c9fc0](https://github.com/Ariestar/sivtr/commit/68c9fc0))
- Copy and diff live under `memory/`; CLI write/read domains split; single-buffer TUI path removed. ([c57375e](https://github.com/Ariestar/sivtr/commit/c57375e), [fa230ec](https://github.com/Ariestar/sivtr/commit/fa230ec))
- TUI modules split into `content/` and `workspace/`; legacy `content_*` / `workspace_search` path aliases dropped. ([de6d32c](https://github.com/Ariestar/sivtr/commit/de6d32c), [0abf4fd](https://github.com/Ariestar/sivtr/commit/0abf4fd))
- Search defaults to dialogue parts; MCP part kind dropped (thinking kept). ([0dc7b44](https://github.com/Ariestar/sivtr/commit/0dc7b44))
- Agent providers regrouped under `agents/` with shared skeletons; MSRV documented as 1.91. ([5ccb033](https://github.com/Ariestar/sivtr/commit/5ccb033), [1789aed](https://github.com/Ariestar/sivtr/commit/1789aed))

### Fixed

- Filter more agent startup / interrupt / IDE-wrapper noise so those blocks are not standalone chat turns. ([00e35f3](https://github.com/Ariestar/sivtr/commit/00e35f3))
- Harden remote peer dials: refresh observed endpoints after success, retry by EndpointId when stored addresses fail, quieter handshake logs; `serve status` reports mode / relay / direct address counts. ([9f796f3](https://github.com/Ariestar/sivtr/commit/9f796f3))
- Browse: keep original structure markers when collapsing; count identical markers across an IO section; stop blanking Ready during meta expand. ([43d4b6a](https://github.com/Ariestar/sivtr/commit/43d4b6a), [364c490](https://github.com/Ariestar/sivtr/commit/364c490), [1a6b173](https://github.com/Ariestar/sivtr/commit/1a6b173))

### Performance

- Session list stays meta-only; dialogue bodies materialize sparsely via `SlidingPane` / `Pane` contract; borrowed `PaneInput` and criterion/dhat benches for sparse materialize. ([abd28fa](https://github.com/Ariestar/sivtr/commit/abd28fa), [afe5a99](https://github.com/Ariestar/sivtr/commit/afe5a99), [fdc68df](https://github.com/Ariestar/sivtr/commit/fdc68df), [60703f2](https://github.com/Ariestar/sivtr/commit/60703f2))

### Documentation

- README is MCP-first; docs aligned with share/invite, provider list, and `ws` origins. ([a309b1c](https://github.com/Ariestar/sivtr/commit/a309b1c), [50330d8](https://github.com/Ariestar/sivtr/commit/50330d8), [9a86004](https://github.com/Ariestar/sivtr/commit/9a86004), [107ff76](https://github.com/Ariestar/sivtr/commit/107ff76))

### 新增

- 新增 Grok agent provider（解析 `~/.grok`），MCP install/uninstall 统一到 host registry，所有 provider 共用配置路径。([f88364e](https://github.com/Ariestar/sivtr/commit/f88364e))
- OpenClaw 从 per-agent SQLite（`session_entries` + `transcript_events`）读取，JSONL 作遗留回退；Cursor 接入真实 agent-transcript JSONL。([88a7e64](https://github.com/Ariestar/sivtr/commit/88a7e64), [c43306c](https://github.com/Ariestar/sivtr/commit/c43306c))
- Hermes 主存储改为 `state.db`（JSONL 残余）；workspace 过滤保留无 cwd 的 unbound session。([e1024d4](https://github.com/Ariestar/sivtr/commit/e1024d4))
- 结构化 part：skill / thinking / MCP 通道记为 `<:channel:…:>` 标记；阅读模式折叠为 marker 行，raw 展开完整载荷。([84af70d](https://github.com/Ariestar/sivtr/commit/84af70d), [6f5d34c](https://github.com/Ariestar/sivtr/commit/6f5d34c), [1c39dba](https://github.com/Ariestar/sivtr/commit/1c39dba))
- Browse TUI 成为产品入口（`sivtr` / hotkey / pick）：多源 `query_many`、非阻塞源加载、help/footer 键位表驱动。([25641d0](https://github.com/Ariestar/sivtr/commit/25641d0), [1a2cfe5](https://github.com/Ariestar/sivtr/commit/1a2cfe5), [48b20d3](https://github.com/Ariestar/sivtr/commit/48b20d3))
- 内容区拆成独立 Input / Output 半窗（Tab 切换、鼠标命中、自由拖选 / Ctrl 拖块选）；焦点偏置高度，源列表聚焦时纵向展开。([d9e1e16](https://github.com/Ariestar/sivtr/commit/d9e1e16), [2ead582](https://github.com/Ariestar/sivtr/commit/2ead582), [b0cd671](https://github.com/Ariestar/sivtr/commit/b0cd671))
- 视口滑动窗口加载 session：meta 按可见行 + prefetch 增长；body 只保留焦点 / 多选 / 邻居，滚离即回收。([3c7d118](https://github.com/Ariestar/sivtr/commit/3c7d118))
- 配置级 `&alias` 源展开（`alias set|list|remove`），加载前解析短别名。([0b8035b](https://github.com/Ariestar/sivtr/commit/0b8035b), [94087f1](https://github.com/Ariestar/sivtr/commit/94087f1))
- MCP install 用 `-p/--provider` 选 host（默认检测已装）；OpenClaw host 写入 `~/.openclaw/openclaw.json`。([45a5635](https://github.com/Ariestar/sivtr/commit/45a5635), [c43306c](https://github.com/Ariestar/sivtr/commit/c43306c))
- doctor 检查 `data_dir` 下 session 存储，可安装 `sivtr-memory` skill；`--fix` 合并唯一日志并清理遗留 workspace key 重复项。setup 缺失时也会装 skill。([4ac7e0d](https://github.com/Ariestar/sivtr/commit/4ac7e0d), [919bbaa](https://github.com/Ariestar/sivtr/commit/919bbaa))
- copy agent 入口走 registry（无 per-provider clap 变体）；doctor 拆分 agent-host 检测与真实 sivtr MCP 注册。([1789aed](https://github.com/Ariestar/sivtr/commit/1789aed))

### 变更

- `WorkRef` 改为 `scope + path + at`（如 `desk:codex/abc/5/i/2`）；移除 `Local`/`Remote` 与 `origin:body` 兼容层。mount 别名仍是 named scope。([a8f2eff](https://github.com/Ariestar/sivtr/commit/a8f2eff))
- share invite 独立：裸 `sivtr share` 只建 share；发 key 用 `sivtr share invite <name>`。([6cd1569](https://github.com/Ariestar/sivtr/commit/6cd1569), [01273b6](https://github.com/Ariestar/sivtr/commit/01273b6))
- 移除 `sivtr use` / workspace `context.toml`；只保留显式 `desk:` / `&alias:`。([94087f1](https://github.com/Ariestar/sivtr/commit/94087f1))
- MCP install 不再接受 `auto` 或 `-t/--target`；改用 `-p/--provider` 或默认检测。([45a5635](https://github.com/Ariestar/sivtr/commit/45a5635))
- workspace origin CLI 重命名为 `sivtr ws`（原 `wb`）。([a309b1c](https://github.com/Ariestar/sivtr/commit/a309b1c))
- 本地与远程 share 共用 `query(source, Filter)`；remote 只转发 Query，在 share root 上套同一过滤。([68c9fc0](https://github.com/Ariestar/sivtr/commit/68c9fc0))
- copy / diff 归入 `memory/`；CLI 写/读域拆分；去掉单缓冲 TUI 路径。([c57375e](https://github.com/Ariestar/sivtr/commit/c57375e), [fa230ec](https://github.com/Ariestar/sivtr/commit/fa230ec))
- TUI 拆成 `content/` 与 `workspace/`；删除 `content_*` / `workspace_search` 路径别名。([de6d32c](https://github.com/Ariestar/sivtr/commit/de6d32c), [0abf4fd](https://github.com/Ariestar/sivtr/commit/0abf4fd))
- search 默认 dialogue part；去掉 MCP part kind（保留 thinking）。([0dc7b44](https://github.com/Ariestar/sivtr/commit/0dc7b44))
- agent provider 归入 `agents/` 共享骨架；MSRV 记为 1.91。([5ccb033](https://github.com/Ariestar/sivtr/commit/5ccb033), [1789aed](https://github.com/Ariestar/sivtr/commit/1789aed))

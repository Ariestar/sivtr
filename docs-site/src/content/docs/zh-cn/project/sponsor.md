如果 sivtr 帮你省了时间，可选的赞助会帮助项目持续维护。

## 微信赞赏

请用微信扫描下方二维码：

![微信赞赏码](/sponsor/wechat-qr.png)

> 若图片未显示，说明维护者尚未上传赞赏码资源。也可在可用时使用 [GitHub Sponsors](https://github.com/sponsors/Ariestar)，或开 issue 索取当前赞赏码。

## 其它支持方式

- Star 并分享 [github.com/Ariestar/sivtr](https://github.com/Ariestar/sivtr)
- 提交清晰的 bug 报告与文档修正
- 贡献代码 — 见 [CONTRIBUTING.md](https://github.com/Ariestar/sivtr/blob/main/CONTRIBUTING.md)

谢谢。
